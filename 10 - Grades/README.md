## ğŸ§  If Statement

An `if` statement is used to test a condition for truth:

- If the condition evaluates to `True`, the code inside the `if` block is executed.
- If the condition evaluates to `False`, that block is skipped.

```python
if condition:
  # code inside
```

### Example:

```python
grade = 70

if grade > 60:
  print("You passed!")
```
